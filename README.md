## Avo market
Avo market is a web application which can show avocado's market trend.

### Live app link
https://avomarket.netlify.com

### App built with
* React
* HTML
* CSS
* Node.js

### Function
* Sign up and Login with auth0
* Data visualization
* Amazon lex Chatbot
